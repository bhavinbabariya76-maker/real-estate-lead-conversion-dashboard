# real-estate-lead-conversion-dashboard
Lead conversion and marketing performance dashboard for real estate businesses using Python.
Real Estate Lead Conversion Dashboard

Project Overview

This project analyzes real estate marketing and lead generation performance using Python, Pandas, and Matplotlib.

The objective is to identify which marketing channels generate the highest quality leads and the best return on investment.

Dataset

The dataset contains information about:

* Lead Source
* Number of Leads
* Number of Bookings
* Marketing Spend

Key Metrics

Conversion Rate

Measures how many leads convert into bookings.

Cost Per Booking

Calculates the marketing cost required to generate one booking.

Total Bookings

Tracks performance across different lead sources.

Business Questions Answered

* Which lead source has the highest conversion rate?
* Which marketing channel generates the most bookings?
* Which source provides the lowest cost per booking?
* Which channel delivers the highest return on marketing spend?

Tools Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

Key Insights

* Identified the most effective lead generation channels.
* Measured conversion performance across sources.
* Compared marketing costs and booking outcomes.
* Provided recommendations for budget allocation.
